# プロジェクト管理

このフォルダは、アクティブなプロジェクトとアイデアを管理します。

## フォルダ構造

- `active/` - 進行中のプロジェクト
- `ideas/` - プロジェクトアイデア、やりたいことリスト

## ワークフロー

1. 新しいプロジェクトアイデアは`ideas/`に追加
2. プロジェクトを開始したら`active/`に移動
3. プロジェクトが完了したら`05_Archive/projects/`に移動
