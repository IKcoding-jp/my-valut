# Obsidian x Cursor 統合ルール

このファイルは、ObsidianとCursor IDEの統合に関するプロジェクト固有のルールを定義します。

## ファイル構造

- Obsidianの設定ファイル（`.obsidian/`）はCursorで編集しない
- マークダウンファイルはObsidianとCursorの両方で編集可能
- コードファイルは主にCursorで編集

## 命名規則

- 文献ノート: `[著者名]-[タイトル].md`
- 永続ノート: `YYYY-MM-DD-[タイトル].md` または `[タイトル].md`
- プロジェクト: `[プロジェクト名]/README.md`

## ワークフロー

1. Obsidianでノートを書く
2. プロジェクト関連のコードはCursorで編集
3. 両方のツールで同じプロジェクト構造を使用
