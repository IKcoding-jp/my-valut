# 私について

## 仕事について

### これまでの経緯

中学は学業や部活、先生や友人関係で苦労し、精神的に疲労したことで不登校になりました。高校も通信制の高校で、ほとんど家で過ごしてました。

卒業後、通信制の大学に行きましたが、勉強ができず、単位が取れず、強制退学となり大学中退しました。

アルバイト経験や人とかかわる機会が少なかったため、仕事をする勇気が持てませんでした。なので、**2022年から2023年はニート**でした。そこでうつ病を発症してしまい、病院に通うことになりました。

なんども自殺しようとしましたが、生きることを諦められませんでした。

### 転機と現在の仕事

2024年に就労移行支援施設「**ぱるスイッチ**」（新所沢）に1年ほど通い、現在は障害者向けの株式会社スタートラインが運営する「**BYSN**」というコーヒー豆のハンドピックや焙煎をする仕事についています。

今仕事をすることができていますが、正直にいうと生活のためであり、大変ではあります。しかし、障害者向けなので配慮もしてもらえるし、ノルマや制限はないし、楽な仕事です。

少々退屈さはありますが、少しずつやりがいを感じたり好きに感じることが増えました。

### この仕事を選んだ理由

この仕事を選んだのは、**とにかくお金をすぐ稼ぐ必要があった**からです。

本当はもっと時間をかけて仕事を選びたかったですが、弟が18歳になり仕事に就くことで、私たちは生活保護の受給をやめる必要がありました。同じ世帯だったので、最低生活費を上回ってしまったんですね。

なので、生活費を稼ぎ、家に入れないと家族の生活が成り立たないような状況になりました。

| 時期 | 出来事 |
|------|--------|
| 2025年1月 | 生活保護の受給終了を知る |
| 2025年3月 | 就活開始 |
| 2025年5月 | BYSNで就労開始 |
| 2026年1月（現在） | 引き続き勤務中 |

## なぜITなのか

### きっかけ

最初は「**まいカゴ**」というAndroid向けアプリをCursorなどで**バイブコーディング**で作ったことがきっかけです。

🔗 [まいカゴ - Google Play](https://play.google.com/store/apps/details?id=com.ikcoding.maikago)

2025年7月頃、バイブコーディングでビジュアルノベルゲームを作ったという記事を見て、自分もやってみようと思いました。

### ローストプラスの開発

今私が働いている場所での**業務改善アプリ**を作ってみようと思い、生まれたのが「**ローストプラス**」です。

- 最初はFlutterで作っていた
- まいカゴをストアにリリースする大変さを知り、手軽にデプロイ可能な**Next.js**を採用
- Webアプリ化して作り直した

> あとでGitHubにあるリポジトリを見せたいと思います。

### IT学習への決意

これらのアプリを作ったことで、自分がソフトウェアエンジニアとして社会に貢献したり人の役に立てる可能性を感じました。

また、バイブコーディングの限界や、知識のないエンジニアリングの危険性、自分が求めているものを作れないこと、利用者からの信頼性のなさなども考慮し、**基礎的な学習こそAI時代に必要**だと強く感じました。

#### 取得済み・勉強中の資格

| 資格 | 状況 |
|------|------|
| 生成AIパスポート | ✅ 取得済み |
| ITパスポート | ✅ 取得済み |
| G検定 | ✅ 取得済み |
| 基本情報技術者試験 | 📚 勉強中 |

これが、私がITエンジニアという職業に強く興味を持ったきっかけになりました。

> 自分は学がなく、地頭もいいほうではないですが、**継続的な努力とあきらめない力**を信じて、エンジニアになるという夢に向かって頑張っています。
