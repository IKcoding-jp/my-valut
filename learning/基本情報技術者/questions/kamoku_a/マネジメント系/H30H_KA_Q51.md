## 問題
図のアローダイアグラムにおいて，プロジェクト全体の期間を短縮するために，作業A～Eの幾つかを1日ずつ短縮する。プロジェクト全体を2日短縮できる作業の組合せはどれか。![51.png/image-size:294×89](https://www.fe-siken.com/kakomon/30_haru/img/51.png)

## 選択肢
- ア: A，C，E
- イ: A，D
- ウ: B，C，E
- エ: B，D

## 正解
エ

## 解説
まず図のアローダイアグラムにおけるクリティカルパスを求めます。以下、各作業を"作業名(作業日数)"で表記しています。
クリティカルパスは「B→C→D」、プロジェクト全体の最短完了日数は「9日」とわかります。
作業を短縮した後のクリティカルパスを選択肢ごとに再計算していきます。
したがって、プロジェクト全体を2日短縮できる作業の組合せは「B，D」となります。

### 各選択肢の解説
A(6) → D(2)＝8日
B(3) → C(4) → D(2)＝9日
B(3) → E(4) → F(1)＝8日
- ア: A・C・Eを1日ずつ短縮すると、A(5) → D(2)＝7日B(3) → C(3) → D(2)＝8日B(3) → E(3) → F(1)＝7日最短完了日数は8日で短縮可能日数は1日です。
- イ: A・Dを1日ずつ短縮すると、A(5) → D(1)＝6日B(3) → C(4) → D(1)＝8日B(3) → E(4) → F(1)＝8日最短完了日数は8日で短縮可能日数は1日です。
- ウ: B・C・Eを1日ずつ短縮すると、A(6) → D(2)＝8日B(2) → C(3) → D(2)＝7日B(2) → E(3) → F(1)＝6日最短完了日数は8日で短縮可能日数は1日です。
- エ: B・Dを1日ずつ短縮すると、A(6) → D(1)＝7日B(2) → C(4) → D(1)＝7日B(2) → E(4) → F(1)＝7日最短完了日数は7日で短縮可能日数は2日です。
