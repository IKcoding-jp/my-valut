## 問題

次のプログラムは、整数型の配列 `data` の中から目的の値 `target` を線形探索し、見つかった場合はその添字を、見つからなかった場合は `-1` を返す。プログラム中の【　】に入れる正しい答えを、選択肢の中から選べ。

```
○整数型: linearSearch(整数型の配列: data, 整数型: target)
  整数型: i
  for (i を 0 から data の要素数 - 1 まで 1 ずつ増やす)
    if (【　　】)
      return i
    endif
  endfor
  return -1
```

配列 `data` が `{5, 12, 3, 8, 1}` で、`target` が `8` のとき、このプログラムの戻り値として正しいものはどれか。

ただし、配列の添字は 0 から始まるものとする。

## 選択肢
- ア: 2
- イ: 3
- ウ: 4
- エ: -1

## 正解
イ

## 解説

### トレース過程

配列 `data = {5, 12, 3, 8, 1}`、`target = 8` で実行する。

【　】に入るのは `data[i] = target`（配列の i 番目の要素が目的の値と等しいか）である。

| ループ回数 | i の値 | data[i] | data[i] = target (8) ? | 動作 |
|-----------|--------|---------|------------------------|------|
| 1回目 | 0 | 5 | 5 ≠ 8 → 偽 | 次へ |
| 2回目 | 1 | 12 | 12 ≠ 8 → 偽 | 次へ |
| 3回目 | 2 | 3 | 3 ≠ 8 → 偽 | 次へ |
| 4回目 | 3 | 8 | 8 = 8 → 真 | return 3 |

`i = 3` のとき `data[3]` が `8` と一致するので、`return 3` が実行される。

### 正解の理由

配列の先頭（添字 0）から順番に `target` の値 `8` と比較していく。添字 3 の位置に `8` があるため、戻り値は `3` となる。よって正解は **イ** である。

### アルゴリズムのポイント
- 線形探索は配列の先頭から末尾まで1つずつ順番に調べる、最も基本的な探索アルゴリズムである
- 見つかった瞬間に `return` で即座に抜ける点がポイント
- 最悪の場合（末尾にある or 見つからない）、配列の全要素を調べるため計算量は O(n)
- 配列がソートされている必要はない
