# 朝のチェックイン

## ワークフロー

1. **日付と試験カウントダウンを表示**
   - 今日の日付（YYYY-MM-DD、曜日）
   - 基本情報技術者試験（2026-02-21）までの残り日数
   - コーヒーマイスター実技講習（2026-05予定）までのおおよその残り日数

2. **デイリージャーナルを作成**
   - パス: `journal/YYYY/MM/YYYY-MM-DD.md`
   - テンプレート: `journal/_template_daily.md` を読み取り、日付のプレースホルダーを実際の日付に置換して生成
   - 既に存在する場合はスキップし、その旨伝える

3. **今日の学習計画を確認**
   - `learning/進捗ログ.md` を読み、直近の学習状況を把握
   - `learning/学習計画書_資格同時取得.md` を読み、今日やるべき範囲を提案
   - 試験直前期（残り14日以内）は過去問演習を強く推奨

4. **未処理タスクを確認**
   - `tasks/タスク.md` を読み、未完了タスクがあれば一覧表示
   - `tasks/今日の１日の計画.md` があれば内容を表示

5. **気分・体調の確認**
   - 「今日の気分・体調はどうですか？」と聞く
   - 回答があればジャーナルの該当セクションに記録

## 出力フォーマット

表形式で簡潔にまとめる：

```
## おはようございます！（YYYY-MM-DD 曜日）

| 項目 | 内容 |
|------|------|
| 基本情報技術者試験 | あと X 日 |
| コーヒーマイスター | あと 約X ヶ月 |
| ジャーナル | 作成済み / 既に存在 |

### 今日の学習予定
- ...

### 未処理タスク
- ...
```

## 注意事項

- テンプレート内の `{{date:YYYY-MM-DD}}` や `{{date:ddd}}` は実際の日付・曜日に置換する
- 曜日は日本語で表示（月, 火, 水, 木, 金, 土, 日）
- journalディレクトリの月フォルダ（`journal/YYYY/MM/`）が存在しない場合は作成する
