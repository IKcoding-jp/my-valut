# 読書管理

## ワークフロー

### 読了記録

1. **情報をヒアリング**
   - 書名（必須）
   - 著者（必須）
   - 読書の目的（任意）
   - 主な学び（任意、後から追記可）
   - 印象に残った点（任意）
   - 実践したいこと（任意）

2. **読書感想ファイルを作成**
   - パス: `reading/読書感想_[書名].md`
   - 既存の読書感想ファイルのフォーマットに従う:
     ```
     ---
     tags:
       - reading/感想
       - status/完了
     ---

     # 読書感想：『[書名]』

     **著者**：[著者名]

     ---

     ## 読書の目的
     ...

     ## 主な学び
     ...

     ## 印象に残った点
     ...

     ## 実践したいこと
     ...
     ```

3. **購入本ランキングを更新**
   - `reading/購入本_読書順ランキング.md` に該当書籍があれば、読了マーク等で更新

4. **確認を表示**
   ```
   | 項目 | 内容 |
   |------|------|
   | 書名 | ... |
   | 著者 | ... |
   | 感想ファイル | reading/読書感想_[書名].md に作成済み |
   | ランキング | 更新済み / 該当なし |
   ```

### 次に読む本の確認

ユーザーが「次は何を読む？」と聞いた場合:
- `reading/購入本_読書順ランキング.md` を読み、未読の最上位を提案

## 注意事項

- 感想の内容が薄くても問題ない。後から追記できることを伝える
- 既存の読書感想ファイルのスタイルに合わせる
