# コンテンツ記録

## ワークフロー

1. **情報をヒアリング**
   - 作品名（必須）
   - カテゴリ（アニメ / 映画 / ゲーム / ドラマ / その他）
   - スコア（5.0満点、0.5刻み）（必須）
   - サブカテゴリ（アニメなら TV / Movie 等）
   - 簡単な感想（任意）

2. **見た作品リストに追加**
   - `entertainment/見た作品・ゲームリスト.md` の該当カテゴリ・サブカテゴリのテーブルに行を追加
   - テーブルはスコア降順 → タイトル昇順で並べる

3. **見たい作品リストから削除**
   - `entertainment/見たい作品リスト.md` に同じ作品名があれば削除

4. **確認を表示**
   ```
   | 項目 | 内容 |
   |------|------|
   | 作品名 | ... |
   | カテゴリ | ... |
   | スコア | X.X / 5.0 |
   | 記録先 | 見た作品リストに追加済み |
   | 見たいリスト | 削除済み / 該当なし |
   ```

## 注意事項

- 既存のテーブルフォーマットに合わせる（`| タイトル | スコア |`）
- 同じ作品名が既に存在する場合はスコアの上書きを提案
- 感想がある場合、ジャーナルの自由メモセクションへの記録を提案
